## Task 1: Fix Status Update Bug (CRITICAL)

**Problem:** `ExpandableOrderCard.onStatusChange` callback takes only `(orderId: string)` but `handleKanbanStatusChange` in `OrdersListScreen` expects `(orderId: string, targetStatus: string)`. The card calls `onStatusChange(order.id)` without passing the target status, so `targetStatus` arrives as `undefined` and the API silently fails.

**Root cause:** In `KanbanBoard3Col.tsx` line 266, the card receives:
```tsx
onStatusChange={() => handleStatusButtonPress(order.id, config.key)}
```
This wraps the call correctly — BUT `ExpandableOrderCard` never calls the passed `onStatusChange` with the right params. Lines 423, 430, 438 all call `onStatusChange(order.id)` which invokes that wrapper with no args, but the wrapper ignores arguments anyway and calls `handleStatusButtonPress(order.id, config.key)` correctly.

**The REAL bug:** The wrapper `() => handleStatusButtonPress(...)` ignores whatever ExpandableOrderCard passes. But `handleStatusButtonPress` for the 'complete' column calls `onStatusChange(orderId, 'preparing')` — this moves BACKWARD (complete → preparing), but the button on an expanded card in the 'active' column should move FORWARD (active → ready). The column-to-status mapping may be wrong.

**Files:**
- Modify: `src/components/orders/ExpandableOrderCard.tsx:48` (callback signature)
- Modify: `src/components/orders/ExpandableOrderCard.tsx:423,430,438` (callback calls)
- Modify: `src/components/orders/KanbanBoard3Col.tsx:266` (remove wrapper, pass directly)

**Step 1: Fix ExpandableOrderCard callback signature**

In `src/components/orders/ExpandableOrderCard.tsx`, change line 48:
```tsx
// FROM:
onStatusChange: (orderId: string) => void;
// TO:
onStatusChange: (orderId: string, targetStatus: string) => void;
```

**Step 2: Add nextStatus calculation to ExpandableOrderCard**

After the existing `nextAction` logic (around line 85-100), add a computed `nextStatus` that maps column to the correct next status:
```tsx
const nextStatus = (() => {
  switch (column) {
    case 'new': return 'confirmed';
    case 'active': return 'ready';
    case 'complete': return ''; // no forward from complete
    default: return '';
  }
})();
```

**Step 3: Update all onStatusChange calls to pass nextStatus**

Lines 423, 430, 438 — change from:
```tsx
onStatusChange(order.id);
```
To:
```tsx
onStatusChange(order.id, nextStatus);
```

**Step 4: Simplify KanbanBoard3Col callback**

Line 266 in `KanbanBoard3Col.tsx`, change from:
```tsx
onStatusChange={() => handleStatusButtonPress(order.id, config.key)}
```
To:
```tsx
onStatusChange={onStatusChange}
```

This passes the parent's `onStatusChange` directly — the card now sends `(orderId, targetStatus)` itself.

**Step 5: Test**
- Tap the action button on a card in "New" column → should move to "Active"
- Tap the action button on a card in "Active" column → should move to "Complete"
- Drag a card right → should advance status
- Drag a card left → should revert status

## Task 3: Footer Row with Logo

**Problem:** Current footer is basic text-only. Need to match the Neon sister app style: restaurant logo + name on left, status indicators, settings/reload/gear on right. The `locationLogoUrl` prop is already passed from OrdersListScreen but the component ignores it.

**Files:**
- Modify: `src/components/orders/OrdersBottomDock.tsx`

**Step 1: Rewrite OrdersBottomDock**

Replace the entire component with this version that adds Image, logo container, status dot, styled buttons:

```tsx
import React from 'react';
import { View, Text, Image, TouchableOpacity, StyleSheet } from 'react-native';
import { useSafeAreaInsets } from 'react-native-safe-area-context';
import { useTheme } from '../../theme';

interface OrdersBottomDockProps {
  restaurantName: string;
  locationLogoUrl?: string | null;
  viewMode?: string;
  counts?: Record<string, number>;
  isOnline?: boolean;
  printerConnected?: boolean;
  onOpenSettings?: () => void;
  onRefresh?: () => void;
  recall?: {
    enabled: boolean;
    archivedCount: number;
    active: boolean;
    onToggle: () => void;
  };
}

export const OrdersBottomDock: React.FC<OrdersBottomDockProps> = ({
  restaurantName,
  locationLogoUrl,
  isOnline = true,
  printerConnected = false,
  onOpenSettings,
  onRefresh,
  recall,
}) => {
  const { theme: colors } = useTheme();
  const insets = useSafeAreaInsets();

  const statusColor = !isOnline ? '#ef4444' : printerConnected ? '#22c55e' : '#f59e0b';

  return (
    <View style={[styles.dock, { backgroundColor: colors.surface, borderTopColor: colors.border, paddingBottom: Math.max(insets.bottom, 8) }]}>
      {/* Left: Logo + Restaurant Name */}
      <View style={styles.left}>
        <View style={[styles.logoContainer, { borderColor: colors.border }]}>
          {locationLogoUrl ? (
            <Image source={{ uri: locationLogoUrl }} style={styles.logo} resizeMode="contain" />
          ) : (
            <Image source={require('../../../assets/icon.png')} style={styles.logo} resizeMode="contain" />
          )}
        </View>
        <Text style={[styles.name, { color: colors.text }]} numberOfLines={1}>{restaurantName}</Text>
      </View>

      {/* Right: Recall + Refresh + Status dot + Settings gear */}
      <View style={styles.right}>
        {recall?.enabled && (
          <TouchableOpacity style={[styles.btn, { backgroundColor: recall.active ? '#3b82f6' : colors.border }]} onPress={recall.onToggle}>
            <Text style={[styles.btnText, { color: recall.active ? '#fff' : colors.textMuted }]}>
              {recall.active ? 'Exit Recall' : `Recall (${recall.archivedCount})`}
            </Text>
          </TouchableOpacity>
        )}
        {onRefresh && (
          <TouchableOpacity style={[styles.btn, { backgroundColor: colors.border }]} onPress={onRefresh}>
            <Text style={[styles.refreshText, { color: colors.textMuted }]}>↻</Text>
          </TouchableOpacity>
        )}
        <View style={[styles.statusDot, { backgroundColor: statusColor }]} />
        {onOpenSettings && (
          <TouchableOpacity style={[styles.gearBtn, { backgroundColor: colors.border }]} onPress={onOpenSettings}>
            <Text style={[styles.gearText, { color: colors.textMuted }]}>⚙</Text>
          </TouchableOpacity>
        )}
      </View>
    </View>
  );
};

const styles = StyleSheet.create({
  dock: { flexDirection: 'row', alignItems: 'center', paddingHorizontal: 14, paddingVertical: 10, borderTopWidth: 1 },
  left: { flex: 1, flexDirection: 'row', alignItems: 'center', gap: 10, minWidth: 0 },
  logoContainer: { width: 36, height: 36, borderRadius: 8, overflow: 'hidden', borderWidth: 1, backgroundColor: '#ffffff', alignItems: 'center', justifyContent: 'center' },
  logo: { width: 36, height: 36 },
  name: { fontSize: 16, fontWeight: '700', flexShrink: 1 },
  right: { flexDirection: 'row', alignItems: 'center', gap: 10 },
  btn: { paddingHorizontal: 10, paddingVertical: 6, borderRadius: 8 },
  btnActive: { backgroundColor: '#3b82f6' },
  btnText: { fontSize: 12, fontWeight: '600' },
  refreshText: { fontSize: 16, fontWeight: '800' },
  statusDot: { width: 12, height: 12, borderRadius: 6 },
  gearBtn: { width: 40, height: 40, borderRadius: 10, alignItems: 'center', justifyContent: 'center' },
  gearText: { fontSize: 20 },
});

export default OrdersBottomDock;
```
